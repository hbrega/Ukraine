1. Content
This package contains:

	Mk10A.lod
	Mk10A.bmp
	Mk10A Readme.txt
	
	
	
2. Installation 
Unrar all the files in to your Objects/Aircraft folder. Then, edit your plane's _DATA.ini to add or replace the existing seat with this one. 


SeatModelName=Mk10A
SeatPosition=*


* The SeatPosition is the same as for the pilot, so just copy the numbers from it.
