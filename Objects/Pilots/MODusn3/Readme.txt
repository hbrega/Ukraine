
                     WildCat´s  VFA-131 Wildcat pilot                  


This pilot is for FA-18 VFA-131.



                          Installation  
                            

Copy the files to your "...\Objects\Aircraft  folder


Change the name of pilot in the FA-18A_data.ini :


// Crew ---------------------------------------------------------

[Pilot]
SystemType=PILOT_COCKPIT
PilotModelName=PilotVFA-131   <---This line---
Position=0.0,3.52,0.83
SeatModelName=seat_SJU-17
SeatPosition=0.00,3.61,0.4
MinExtentPosition=-0.25, 5.91,-0.20
MaxExtentPosition= 0.25, 6.99, 0.92
CanopyNodeName=canopy
HasArmor=TRUE
ArmorMaterial=STEEL
Armor[FRONT].Thickness=12
Armor[RIGHT].Thickness=12
Armor[LEFT].Thickness=12
Armor[REAR].Thickness=12
Armor[BOTTOM].Thickness=12
